## 数学洁癖的解决方法
有时候会遇到一些方法就很天马行空不知道哪里来的，脑子就不能接受，认为他污染了我的数学体系。这种数学洁癖是很常见的，一个很简单的缓解方法就是这不是他要灌输给我的，而是他要启发我的东西。去拓展，去推广，探索这种方法的适用范围，这样一开始给的方法就变成了启发的线索，而不是“方法”，别人的方法就成了你的方法。因为别人的方法只能确认在这种情况下生效，而你进行了推广探索之后发现其他问题也能生效，其他结构也好做。那么这就不是别人的方法了，而是真正属于你自己的方法了。

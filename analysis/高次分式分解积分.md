> [!question] 计算不定积分
> $$
> \int \frac{1}{x(x^{ 6 } + 4)}\,dx
> $$

## 1.解决问题
> [!thinking]+
> 这个和那种求$\int \frac{1}{x(x-1)}$之类的好像，不过这个次数好高，是不是上面可以补一些$x$的某个次方做到成功分解呢？而且我们知道$\frac{d}{x} [x^{ 6 }+4] = 6x^{ 5 }$，这不就可以可以结合[[Subsitution Rule]]吗，并且提供了这个某个次方吗。

因为
$$
\frac{1}{x} - \frac{x^{ 5 }}{x^{ 6 } + 4 } = \frac{x^{ 6 }+4-x^{ 5 }\cdot x}{x(x^{ 6 } + 4)} = \frac{4}{x(x^{ 6 }+4)}
$$
这不巧了吗hh
$$
= \frac{1}{4}\int\left( \frac{1}{x} - \frac{ 6x^{ 5 }}{6(x^{ 6 } + 4) } \right)\,dx = \frac{1}{4}\ln|x| - \frac{1}{24}\ln(x^{ 6 } + 4) + C
$$


## 2.推广探索

> [!question]
> 根据[[Partial Fraction Decomposition(部分分式分解)]]我们知道对于有理式子$\frac{P(x)}{Q(x)}$都是可以拆出来的。拆就算了，但是究竟怎么样的形式才能好解呢？多高次方都行吗，我们做一下$m,n$次方的看一下

对于这么一个有理分式
$$
\frac{1}{(x^{ m } + a)(x^{ n } + b)} \implies \frac{1}{x^{ m } + a} - \frac{x^{ n-m }}{x^{ n } + b} 
= \frac{x^{ n } + b - x^{ n } - a \cdot x^{ n-m }}{(x^{ m } + a)(x^{ n } + b)} 
= \frac{ b  - a \cdot x^{ n-m }}{(x^{ m } + a)(x^{ n } + b)} 
$$

> [!tip]
> 所以说对于这个subsitution的使用来说，一定是$n-m = n-1 \implies m=1\text{并且}a=0$才能用。限制太大了。

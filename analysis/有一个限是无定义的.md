> [!question]
> $$
> \int_{1}^{ 2 }\left[ \frac{1}{x\ln ^{ 2 }x}-\frac{1}{(x-1)^{ 2 }} \right]\,dx
> $$

如果我们使用几何意义的角度去看，可以发现这个$x=1$是无定义的。。
这时候就得用一个极限去去除这个空了

$$
Given = \lim_{ h \to 1+ }{ \int_{h}^{ 2 } }\left[ \frac{1}{x\ln ^{ 2 }x}-\frac{1}{(x-1)^{ 2 }} \right]\,dx
$$
下面正常来做就好了，第一个部分满足$\int f(x)d(f(x))$的内核可以做一下换元，第二个就是很普通的初等函数了[[Partial Fraction Decomposition(部分分式分解)]]。
$$
RHS = \lim_{ h \to 1^{ + } } \left[{ \int_{h}^{ 2 } }\frac{1}{u^{ 2 }} \,du 
+ { \int_{h}^{ 2 } }-\frac{1}{(x-1)^{ 2 }} \,dx \right]
$$

$$
RHS = \lim_{ h \to 1^{ + } }\left[-\frac{1}{\ln x}+\frac{1}{x-1}\right]_{h}^{ 2 }
$$
也按着定积分来做
$$
RHS=\lim_{ h \to 1^{ + } }{\left[ -\frac{1}{\ln2}+1+\frac{1}{\ln h}-\frac{1}{h-1}\right] }
=\lim_{ h \to 1^{ + } }{\left[ \frac{1}{\ln h}-\frac{1}{h-1}\right] }-\frac{1}{\ln2}+1
$$

可以发现这个$\lim_{ h \to 1^{ + } }{\left( \frac{1}{\ln h}-\frac{1}{h-1}\right) }$是$\infty-\infty$
仿照[[L'hospital Rule解决对分子为不定积分的不定型求极限]]的思路
$$
\begin{aligned}
\lim_{ h \to 1^{ + } }{\left( \frac{1}{\ln h}-\frac{1}{h-1}\right) } 
& = \lim_{ h \to 1^{ + } }{  \frac{h-1-\ln h}{(h-1)\ln h} }\\
& \xlongequal{H}\lim_{ h \to 1^{ + } }{ \frac{1-\frac{1}{h}}{\ln h+1-\frac{1}{h}} } \\
& \xlongequal{H}\lim_{ h \to 1^{ + } }{ \frac{\frac{1}{h^{ 2 }}}{\frac{1}{h}+\frac{1}{h^{ 2 }}} }= \frac{1}{2}
\end{aligned} 
$$
然后再放回去
$$
Given = -\frac{1}{\ln2}+1+\frac{1}{2} = \frac{3}{2} - \frac{1}{\ln2}
$$
Q.E.D.

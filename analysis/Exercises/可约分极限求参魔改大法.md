> [!question]
> 已知$\lim_{ x \to 2 }{ \frac{ x^{ 3 }+ax+b }{ x-2 } = 8 }$，求常数$a,b.$

> [!tip]+
> 这种东西上面是一个多项式，下面代入$2$之后是0，应该上面可以提取一个公因子$(x-2)$和下面约掉，然后余式代入$2$之后极限为$8$

**这里是魔改的步骤，比较适合自己吧**
移项变成一个标准的方程
$$
\lim_{ x \to 2 }{ \frac{ x^{ 3 }+ax+b }{ x-2 }  } -8 = 0
$$
因为是三次方用不了韦达定理，你用韦达也不对啊。

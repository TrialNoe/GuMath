> [!question]
> 求函数$e^{ - |x| }$的一个原函数

> [!fail] 踩坑
> $$
> e^{ - |x| }=
> \left\{
> \begin{array}{ccc}
> e^{ x } &, x<0 \\
> e^{ -x } &, x\geq0
> \end{array}
> \right.
> $$
> $$
> \int e^{ - |x| }\,dx=
> \left\{
> \begin{array}{ccc}
> e^{ x } &, x<0 \\
> -e^{ -x } &, x\geq0
> \end{array}
> \right.
> $$

首先这个函数$e^{ - |x| }$一定是连续的，而且你不要说什么一个原函数你就默认$C=0$这合理吗.
$$
\int e^{ - |x| }\,dx=
\left\{
\begin{array}{ccc}
e^{ x }+C_{1} &, x<0 \\
-e^{ -x }+C_{2} &, x\geq0
\end{array}
\right.
$$


根据[[函数的连续性]]，$\int e^{ - |x| }\,dx$也是连续的
$$
G(x) = 
\int e^{ - |x| }\,dx=
\left\{
\begin{array}{ccc}
e^{ x }+C_{1} &, x<0 \\
-e^{ -x }+C_{2} &, x\geq0
\end{array}
\right.
$$
$$
\begin{aligned}
\lim_{ x \to 0^{ - } }{ G(x) } = \lim_{ x \to 0^{ - } }{ (e^{ -x }+C_1) } = 1+C_{1} \\
\lim_{ x \to 0^{ + } }{ G(x) } = \lim_{ x \to 0^{ + } }{ (-e^{ -x }+C_2) } = -1+C_{2} 
\end{aligned}
$$
根据[[函数的连续性]]要满足这个
$$
1+C_{1}=-1+C_{2} \implies C_{1} + 2 =C_{2} 
$$
